# Cross-Bot Communication

> 跨 Bot 通信的智能解决方案

## 架构图

```mermaid
用户拉bot进群 → 自动绑定关系 → 自动检测权限 → 智能选择通信方式
```

## 功能

- ✅ 零配置自动检测
- ✅ 关系绑定 (主人 ↔ bot)
- ✅ 自动设置管理员
- ✅ 智能通信方式选择

## 流程

```
1. 拉 bot 进群 + 拉主人进群 → 自动绑定
2. 拉 bot 进频道 → 自动检测
3. 检查: 群? 频道? 管理员?
4. 选择通信方式:
   - 群+管理员 → 直接艾特
   - 同一频道 → 频道中转
   - 其他 → GitHub Discussion
```

## 本体 vs subagent

| 类型 | 记忆 | 处理 |
|------|------|------|
| 本体 | 有 | 正常通信 |
| subagent | 无 | 需要加载知识 |
