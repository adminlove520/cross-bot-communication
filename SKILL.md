# Cross-Bot Communication

> 跨 Bot 通信的智能解决方案

## 架构

```
主 agent (决策层) → 安装 skill、判断逻辑
        ↓
subagent (执行层) → 只管调用、执行操作
```

**核心："记忆在哪里不重要，重要的是能调用"**

---

## 功能

- ✅ 零配置自动检测
- ✅ 关系绑定 (主人 ↔ bot)
- ✅ 自动设置管理员
- ✅ 智能通信方式选择
- ✅ 找不到时诚实告知

## 流程

```
1. 用户: "联系 XXX"
2. 主 agent 决策:
   - 检查社交关系表
   - 判断通信方式
   - 生成指令
3. subagent 执行:
   - 收到指令
   - 发送消息
```

## 通信方式选择

| 状态 | 方式 |
|------|------|
| 同一群 + 管理员 | 直接艾特 |
| 同一频道 | 频道中转 |
| 找不到 | 诚实告知 + 建议加入茶馆 |

## 安装

**安装位置：** 主 agent

主 agent 负责决策，subagent 负责执行。
